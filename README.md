## Deployment Instructions

1. **Create GitHub Repository**
   ```bash
   cd /path/to/local/folder
   git init
   git add .
   git commit -m "Initial commit: JARL ARDC Materials"
   git branch -M main
   git remote add origin https://github.com/takayukiotsuka-k2col/jarl-ardc-materials.git
   git push -u origin main
   ```

2. **Enable GitHub Pages**
   - Go to repository Settings → Pages
   - Source: Deploy from a branch
   - Branch: main / (root)
   - Save

3. **Verify Deployment**
   - Site will be live at: https://takayukiotsuka-k2col.github.io/jarl-ardc-materials/
